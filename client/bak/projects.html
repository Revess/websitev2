  • Smartlappen

    For the compagny SETUP I created a system for their workshop.

    SETUP is an Utrecht-based medialab, their main goal is to educate a wide audience, providing them with the tools necessary to design this brave new world, and infuse it with human values and new-found creativity.

    In the project Tranen Op De Dansvloer SETUP created an AI algorithm, using GPT2, to write Dutch Smartlappen. Together with this algorithm they wrote a smartlap themselves named Tranen Op De Dansvloer.

    SETUP has comeup with the idea to create a workshop using the algorithms they wrote their smartlappen with. With the Jij & AI workshop you can try the algorithm for yourself. For this workshop I created the backend network system and retrained the AI for better performance with multiple users. Since with the workshop there are multiple users had to access Tensorflow I created a multi-threaded workload where each task is quickly processed on the GPU of the central server. The users could connect to the server using a mobile device that is connected to the same network and could control the AI in their own seperate session. With the use of Node.js as a middle layer the users were able to connect their data seamlessly to the Python backend.

  • Archeology & AI

    A multidiciplinary project where we innovate archeological research.

    In collaboration with the Archeological Work Group Netherlands and Lars Bijleveld I do research on automatization to identify of historical findings. With Artificial Intelligence we want to create tools for archeologists so they can focus on what is important, the story behind the objects rather than the administrative tasks like; indexing and analyzing.

    By using customized the latest and greatest image recognization algorithm the AI will be able to distinguise clearly between different archeological findings. The training data is made up of real archeological findings, 2D sketches of the original shards and 3D moddeled pots that are virtually broken and then photographed.
